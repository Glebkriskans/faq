# Горячие клавиши:
#### Коппировать: 
```bash
Ctrl+C
```
#### Вставить: 
```bash
Ctrl+V
```
#### Сохранить: 
```bash
Ctrl+S
```
#### Вырезать: 
```bash
Ctrl+X
```
#### Отмена: 
```bash
Ctrl+Z
```
# Работа с git
### Скачать проект:
```bash
git clone <url>
```
### Учесть все файлы:
```bash
git add .  
```
### Сохранить коммит:
```bash
git commit -m "..." 
```
### Отправить в облако:  
```bash
git push
```
